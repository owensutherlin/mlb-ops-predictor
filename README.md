# MLB OPS Predictor
### EAS 510 — Spring 2026 | Owen Sutherlin

A machine learning pipeline that predicts MLB hitter OPS for the 2025 season
using three years of prior performance data (2022–2024). The year window is
updatable, meaning the model can be simply changed to train on 2015-2017 then
predict 2018, for example. The project combines traditional batting statistics 
with modern Statcast metrics to evaluate three models: Linear Regression, 
Random Forest, and XGBoost.

## Quickstart

### 1. Clone the repository
```bash
git clone https://github.com/owensutherlin/mlb-ops-predictor.git
cd mlb-ops-predictor
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Download the data
```bash
PYTHONPATH=. python data/download_data.py
```
This pulls 11 seasons (2015–2025) of batting data directly from
Baseball Savant. Runtime is approximately 2–3 minutes on first run.

### 4. Run the full experiment
```bash
PYTHONPATH=. python main.py
```
This trains all three models, evaluates them on 2018, prints metrics
to the console, and saves all plots and CSVs to the `outputs/` folder.

## Data

Data is sourced from **Baseball Savant** via direct CSV export. Each season
includes the following features per player:

**Traditional:** AVG, OBP, SLG, OPS, BB%, K%

**Statcast:** Exit Velocity (avg), Hard Hit %

**Engineered:** 3-year mean and year-over-year delta for each feature, player age

Players must have a minimum of 250 plate appearances in each of the three
input seasons to be included. This yielded **156 qualifying players** for
the 2015–2017 → 2018 training window.

---

## Models

### Linear Regression (Baseline)
Standard OLS regression with StandardScaler preprocessing. Provides an
interpretable baseline and coefficient-based feature importance.

### Random Forest
Ensemble of 200 decision trees with max depth 5 and minimum 5 samples
per leaf. Captures nonlinear relationships between features.

### XGBoost
Gradient boosted trees with 200 estimators, learning rate 0.05, and
max depth 4. Subsampling (0.8) applied at both row and column level
to reduce overfitting.

---

## Results

All three models trained on 2022–2024 data and evaluated on 2025 OPS.
Performance measured using Mean Absolute Error (MAE) and Root Mean
Squared Error (RMSE):

| Model | MAE | RMSE |
|---|---|---|
| Linear Regression | ~0.052 | ~0.068 |
| Random Forest | ~0.040 | ~0.054 |
| XGBoost | ~0.007 | ~0.009 |


### Key Findings
- **XGBoost** achieved the tightest predicted vs actual OPS values across
  all 156 players, with nearly all predictions within 0.05 OPS
- **Random Forest** outperformed Linear Regression on both MAE and RMSE
- **OPS_mean** was the dominant feature in tree-based models, confirming
  that historical OPS is the strongest predictor of future OPS
- **OBP_delta** was the most important feature for Linear Regression,
  suggesting that trend in on-base ability is a strong linear signal
- Models struggled most with outlier seasons

## Limitations

- **No out-of-sample validation** — models are evaluated on the same
  window used for training. A true holdout set would give a less
  optimistic picture of generalization
- **COVID seasons excluded** — 2020 had fewer than 15
  qualifying players due to the shortened season, making those
  windows unusable for training
- **No injury or off-field data** — player health, rule changes, and
  other external factors are not captured by the feature set
- **Statcast data only available from 2015** — limits the historical
  depth of training data
